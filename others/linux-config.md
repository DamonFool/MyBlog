## 截图软件

|OS版本|截图软件|
|:-|:-|
|Ubuntu16.04|shutter|
|Ubuntu18.04|flameshot|

- 安装flameshot的方法
```shell
sudo apt-get install flameshot
```
- 运行flameshot
```shell
flameshot gui
```

- 快捷键设置
进入系统设置（Setting）>设备（Devices）>键盘（Keyboard）
